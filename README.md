## Installation

* Download the project.
* Open the terminal and navigate to the directory of project ```cd MarvelIVVezeeta```.
* Install Dependencies with Cocoapods ```pod install```.
* Open the workspace ```open MarvelIVVezeeta.xcworkspace```.
